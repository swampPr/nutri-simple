# NutriSimple

A simple project that solves a problem I've had more than once:
- Needing a unified place to track my calories and find recipes based on my goals. With the bonus of quick weather information for deciding the best time to go for a run.

## Project setup

1. Create a PirateWeather key
2. Create a GeoAPIfy key
3. Create a Spoonacular key

**All the necessary secrets are in .env.example file. Add your own keys to the .env.example file and then rename the file to .env**

```bash
$ git clone git@github.com:swampPr/nutri-simple.git
```

```bash
$ npm install
```

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
